# 📘 C Language – Conditionals Practice

This directory is part of my journey to build a strong foundation in **C programming** as a step toward **Embedded Systems and IoT development**.

## 🎯 Purpose

The goal of this section is to develop a deep understanding of **decision-making constructs in C**, which are essential for writing efficient and reliable programs—especially in hardware-oriented environments.

## 📂 Structure

This directory is organized into separate folders based on different types of conditional statements:

* `if` – Basic single-condition checks
* `if-else` – Two-way decision making
* `if-else-if` – Multi-level condition handling
* `nested-if` – Complex decision structures
* `switch` – Multi-choice branching based on exact values
* `ternary` – Compact conditional expressions
* `logical-operators` – Combining multiple conditions
* `bitwise` – Low-level condition checks (important for embedded systems)

Each folder contains multiple practice problems and implementations.

## 🧠 Learning Approach

* Focus on **understanding logic**, not just writing code
* Practice problems with **real-world and embedded-style scenarios**
* Write clean, readable, and structured code
* Gradually improve complexity and efficiency

## 🚀 Goal

To build a solid base in conditional logic that will support:

* Embedded C programming
* Microcontroller-based development
* Real-time decision-making systems
